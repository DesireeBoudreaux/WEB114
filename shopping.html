Week 3 project: If statements and comparison operators

Questions:

Name a situation where you would need to create an "if" statement.

You would use an if statement when your code needs to make a decision based on a condition. For example, if one situation is true, then run a specific block of code. If that condition is not true, then run a different block using else. This allows your program to respond differently depending on what is happening.

When comparing values, in what data type is the result stored?

When comparing values in JavaScript all data tyes result in Boolean, either true or false. This helps to tell if the comparison condition was met or not.
